## Example

  Ensure the following environment variables are set:

  * INDABA_TEST_ENDPOINT
  * INDABA_TEST_CLIENT
  * INDABA_TEST_SECRET

  Then:

  * `npm install`
  * `node example`
  * visit localhost:4000 + login
